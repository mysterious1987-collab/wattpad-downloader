🔑 CÀI ĐẶT KẾT NỐI
Chưa cấu hình
▼
Hướng dẫn tạo PAT (làm 1 lần)
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
Bấm Generate new token → Đặt tên tuỳ ý
Repository access: chọn đúng repo wattpad-downloader
Permissions: Actions → Read and Write · Contents → Read
Copy token → paste vào ô bên dưới
Token lưu trong localStorage trình duyệt, không gửi đi đâu ngoài api.github.com
⬇️ YÊU CẦU TẢI
Nhập URL để bắt đầu
Tên dùng khi tải file từ Artifacts
