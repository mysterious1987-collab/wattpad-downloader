Hướng dẫn tạo PAT (làm 1 lần)
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens
Bấm Generate new token → đặt tên tuỳ ý
Repository access: chọn repo wattpad-downloader
Permissions: Actions → Read and Write  ·  Contents → Read
Copy token → paste vào ô bên dưới
BNS (Bạch Ngọc Sách): tạo Secrets BNS_USERNAME và BNS_PASSWORD trong repo (Settings → Secrets → Actions)
Token lưu trong localStorage, không gửi đi đâu ngoài api.github.com
⬇️ Yêu cầu tải
Nhập URL để bắt đầu
Dán URL trang truyện hoặc URL mục lục (…/muc-luc?page=all) — gửi lên Actions sẽ là URL truyện gốc; script tự mở mục lục đầy đủ. Secrets: BNS_USERNAME / BNS_PASSWORD.
